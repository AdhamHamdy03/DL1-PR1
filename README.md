Image Classification Experiments: Augmentation, Imbalance, and Transfer LearningOverviewThis repository contains a series of experiments investigating three fundamental challenges in computer vision and image classification. The implementations utilize PyTorch and various convolutional neural networks to evaluate different training methodologies.  Project 1: Data AugmentationThis section explores how data augmentation affects the generalization of a SimpleCNN model.  The experiments are conducted on the Scenes 15 dataset, which features indoor and outdoor environments.  Transformations are implemented using the Albumentations library, testing methods like HorizontalFlip, RandomResizedCrop, Rotate, ColorJitter, and InvertedImage.  A custom mixup_data function was implemented to mix two images from the same batch using a Beta distribution.  The combination of HorizontalFlip and RandomResizedCrop yielded the best overall performance and highest test accuracy.  Excessive augmentation and complex combinations ultimately reduced model performance.  Project 2: Class ImbalanceThis study addresses how uneven data distribution impacts a SimpleCNN classifier using the Scenes 15 dataset.  The experiments simulate two main scenarios: a uniform minority setting and a long-tail distribution setting.  A weighted cross-entropy loss function is applied, where minority classes receive larger weights to penalize misclassifications.  Applying the weighted loss strategy significantly improves the accuracy of minority classes compared to the baseline.  The long-tail distribution proved more difficult to handle than uniform imbalance because the model had to compensate for a continuous range of class frequencies.  Project 3: Transfer Learning & Fine-TuningThis project evaluates the adaptation of pre-trained convolutional neural networks to new tasks using the CIFAR-10 dataset.  The study compares multiple ResNet architectures, including ResNet18, ResNet50, and ResNet101.  Different strategies were tested: training from scratch, feature extraction, selective fine-tuning, and full fine-tuning.  Full fine-tuning achieved the best generalization and highest test accuracy by allowing all network parameters to update.  When tested on a reduced four-class dataset, the lightweight ResNet18 achieved higher accuracy and faster training times than the heavyweight ResNet101.  Adding a deeper, more complex classifier head provided limited benefits compared to a simple linear head.
